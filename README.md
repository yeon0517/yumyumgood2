로그인 <br/>
● DB값 검증 <br/>
● ID찾기, PW찾기 <br/>
● 로그인 시 쿠키(Cookie) 및 세션(Session) 생성 <br/>

회원가입 <br/>
● ID 중복 체크 <br/>
● 비밀번호 유효성 검사 <br/>

장바구니 및 결제페이지 <br/>
● 장바구니 재료 선택 <br/>

● 찜한 게시물, 장바구니 게시물 <br/>
● 게시물 작성하기 <br/>

● 구매내역 조회 <br/>
